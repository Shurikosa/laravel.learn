Основні комбінації:
Ctrl + N – Відкрити клас за ім'ям.
Ctrl + Shift + N – Відкрити файл за назвою.
Ctrl + Shift + Alt + N – Відкрити символ (метод або змінну) за назвою.
Ctrl + E – Відкрити нещодавно використані файли.
Ctrl + Shift + E – Список нещодавно закритих файлів.
Ctrl + W – Розширення виділення.
Ctrl + Shift + W – Звуження виділення.
Ctrl + / – Коментувати/розкоментувати рядок.
Ctrl + Shift + / – Коментувати/розкоментувати блок коду.
Ctrl + D – Дублювати рядок або блок коду.
Ctrl + Y – Видалити рядок.
Alt + J – Виділити наступний однаковий фрагмент (мультивибір).
Alt + Shift + J – Виділити всі однакові фрагменти.
Ctrl + Alt + L – Автовирівнювання коду (форматування).
Ctrl + Alt + O – Оптимізувати імпорти.
Навігація:
Ctrl + B – Перейти до визначення.
Ctrl + Alt + B – Перейти до реалізації методу або інтерфейсу.
Alt + F7 – Пошук використання (всі випадки використання).
Ctrl + U – Перейти до батьківського класу або методу.
Ctrl + G – Перейти до рядка.
Ctrl + F12 – Структура файлу (швидкий перегляд методів і полів).
Рефакторинг:
Ctrl + Alt + Shift + T – Відкрити меню рефакторингу.
Shift + F6 – Перейменування.
Ctrl + Alt + V – Витягнути змінну.
Ctrl + Alt + M – Витягнути метод.
Ctrl + Alt + P – Витягнути параметр.
Інше:
Shift двічі – Глобальний пошук по всьому проекту.
Ctrl + Shift + F – Пошук у проекті.
Ctrl + Shift + R – Замінити у проекті.
Alt + Enter – Швидке виправлення (контекстні дії).
